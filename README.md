Bathymetric Plotter

This MATLAB code can be used to create bathymetric maps given latitude longitude and depth data in a csv file.

To use this plotter first ensure that the data file is saved in the same folder as this code and that your depth measurement is in meters
Next change the filename in the line 2 of the code so that it matches your csv
Use Lines 40-43 to set your chart and axis labels
Once run, your plot will generate and you will be able to manipulate and save it
